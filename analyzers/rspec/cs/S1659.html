Why is this an issue?

Declaring multiple variable on one line is difficult to read.

Noncompliant code example

class MyClass
{
  private int a, b; // Noncompliant

  public void Method()
  {
    int c, d; // Noncompliant
  }
}

Compliant solution

class MyClass
{
  private int a;
  private int b;

  public void Method()
  {
    int c;
    int d;
  }
}
